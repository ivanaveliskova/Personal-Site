    Hover Over Images

    See the Pen Description by Ivana Veliskova (@ivanaveliskova) on CodePen.

    I built this hover effect when I was building the first version of my personal site. I had seen a similar hover effect while looking through some code and I wanted to test out some CSS transitions and have a fancy effect on the landing page of my site. Looking back at the code, I was a bit too specific and so there was little option for recycling my code, which I later learned was not a good thing. Now I have improved in making my code more reusable. I also realize how this becomes super inaccessible when it comes to responsive design. Since I learned how to make everything a bit more responsive then I wanted to make sure everything transitioned smoothly. This was difficult to make a simple responsive design because the images are places as background images, therefore there is nothing actually within the <div></div> tags and so there would be no height set, therefore using something like max-height would not work. So I had to physically use media queries in order to make this responsive and remove the hover effect which then didn't look as nice.

    You can view the code and the landing page live on my Github account or if you wish to see it in action you can check it out on my gh-pages.
